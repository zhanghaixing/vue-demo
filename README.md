# vue-demo
vue学习demo
### 实现功能说明
1.通过点击增减btn实现商品的数量的改变并影响小计和总计金额的变化。
2.商品单选和全选之间的联动
3.商品列表的删除和恢复
### 代码总结
1.通过标签移入vue.js,整体思想是直接操控数据实现视图的变化，设置好数据上的关联，响应式改变视图。
2.computed 通过计算属性实现总价在商品选中数目变化时的实时改变
3，通过deleData 记录被删除的数据,实现恢复功能时再将数据导出
